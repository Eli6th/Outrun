Coding Portfolio

Welcome to my coding portfolio! You can type the names of my projects to learn more about them.

Custom-News

Custom-News is a web app that allows users to create their own news feeds by subscribing to their favorite news sources and keywords. It was built using React and Node.js, and utilizes the NewsAPI to fetch and filter articles.

AAPI Month

AAPI Month is a website that celebrates and educates about Asian American and Pacific Islander (AAPI) heritage and culture. It was built using HTML, CSS, and JavaScript, and features a calendar of events and a directory of AAPI-owned businesses.

Project Minuteman

Project Minuteman is a disaster preparedness and response platform. It was built using Python and Django, and allows users to create emergency plans, track their supplies, and receive alerts and updates during a crisis.
